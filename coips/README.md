# Index of Compact Improvement Proposals (CoIPs)

Compact Improvement Proposals (CoIPs) describe proposed changes to the Compact programming language and developer tools.
The process is described in [CoIP-1](coip-0001.md).

CoIPs fall into several categories, which are listed in CoIP-1.
They can have several different statuses as they move through the lifecycle of a CoIP, which are also listed in CoIP-1.
