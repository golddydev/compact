# Agenda and Notes

- [x] **COIP-1 discussion and vote**
  - Kevin will update the proposal based on PR and meeting feedback, and the TSC will vote on the CoIP offline.
  - Decisions:
    - we should require a TSC sponser on each CoIP
    - a reference implemention should be encouraged but not required
    - test cases should be encouraged but not required
- [x] **Compact mechanism for logging events**
  - Dominique will update the proposal based on meeting feedback.
  - Decisions:
    - we should support indexed fields but educate developers about this being an "attractive hazard" in that querying based on specific field values can seriously degrade privacy by limiting the pool of possible interested parties, possibly to a single user.
    - the document should clarify that private data in logged values must be explicitly declared rather than saying that logging does not disclose anything that isn't already disclosed
